Instructions to use SilentSaver web API

USAGE

  1. Input

    • id - Integer Account ID
    • date - Integer Date in format - YYYYMMDD
    • balance - Integer Balance on that date

  2. Output

    • Account ID - Account ID requested
    • Amount to Move - Amount to be moved from Checkin to Saving
    • Balance - Balance available
    • Credit frequency - Average time between two credits
    • Day - Day of month
    • Predicted Expenditure - Amount predicted to be spent in next credit frequency days

  3. Example

    • GET Request - https://silentsaver-164500.appspot.com/api?balance=25000&date=20140521&id=2177
